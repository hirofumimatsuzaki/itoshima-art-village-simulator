# 糸島 芸術村シミュレーター

地方の町に文化施設や生活施設を配置し、アートと経済が循環する村を育てるプロトタイプです。  
海、畑、林のあいだに、糸島らしいのどかな空気を残しつつ、小さな集落を「芸術村」へ育てていきます。

## 概要

このゲームは、一般的な都市開発ではなく、以下を主役にした街づくりを目指しています。

- 文化
- アート
- 交流
- 地方性

現時点では、まず遊べるプロトタイプとして、建物配置と月次シミュレーションを中心に実装しています。

## 現在の機能

- 建物配置
  - ギャラリー
  - カフェ
  - 農場
  - 住居
  - アトリエ
  - 宿泊施設
  - 図書館
  - 工房
- 建物ごとの固有効果
- 月ごとの資金、文化、人口、来訪者、アーティストの変動
- 展示イベント
- 条件を満たしたときのアーティスト出現
- 発展段階
  - 小さな集落
  - 文化の芽生え
  - 芸術の町
  - 芸術村
- 主要ステータスのUI表示

## 遊び方

1. 右側で建物を選びます
2. マップ上の空き地をクリックします
3. `配置する` で建設します
4. `1か月進める` で村の状態を更新します
5. ギャラリーがあると `展示を開く` が使えます

## 動かし方

ブラウザで `index.html` を開くだけで動作します。

## ファイル構成

- `index.html`
  - 画面本体とスタイル
- `sketch.js`
  - p5.js を使った描画、入力、月次シミュレーション、本体ロジック
